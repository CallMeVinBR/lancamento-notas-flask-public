GradeSphere

A sua opção número um para gestão acadêmica!


Este projeto foi desenvolvido com o propósito de aprimorar nossa experiência e competência em desenvolvimento de sites e fins acadêmicos em nossa faculdade.
Agradecemos por usufruir do nosso sistema!


O que é o GradeSphere?

O GradeSphere foi desenvolvido em Python e Flask, usando Django HTML. Esse sistema é open-source e tem todo seu código-fonte no Github. Esse projeto permanecerá hospedado até o dia 21 de Janeiro de 2025! Não se preocupe, ao fim da hospedagem iremos remover todos os dados registrados no sistema.

Este é um sistema (projeto) de gerenciamento acadêmico online com boletins personalizáveis em nomes de matérias, sendo um boletim simples, ou seja:

  • Sem pesos;
  • Colunas predefinidas por períodos;
  • Cálculo de média final automático;
  • Notas e boletim editáveis a qualquer hora e visualização otimizada.

Objetivo de Desenvolvimento Sustentável


Por que optar pelo GradeSphere?

O GradeSphere possui um sistema de segurança, onde todos os dados são criptografados antes da inserção no Banco de Dados para evitar quaisquer tipos de vazamentos ou violações dos mesmos. Os professores não conseguem acessar as telas dos alunos e vice-versa devido ao controle de acesso implementado utilizando sessões Flask, ou falando de uma maneira menos técnica, um controle de acesso que detecta o tipo de conta do usuário e atribui permissões para cada tipo (aluno e professor). Assim, é assegurada a privacidade e segurança dos dados acadêmicos dos alunos e professores.

O GradeSphere possui um sistema de convites tanto para os alunos quanto para os professores, e optamos por atribuir o mesmo para ambos pois, caso um professor convide um aluno errado, esse aluno pode (ou não) recusar o convite. Caso o aluno aceite o convite mesmo sabendo que não é a sua turma, o professor tem a opção de remover ele de sua turma, para evitar constrangimentos ou conflitos no gerenciamento. Porém, o aluno não tem a opção de sair da turma em questão, então tome cuidado e verifique as informações no convite! Caso tenha dúvidas se as informações no convite são verídicas ou não, consulte o seu professor em questão.


Pontos positivos

  • Segurança dos dados do usuário;
  • Privacidade;
  • Fácil acesso;
  • Prevenção contra contas feitas por robôs;
  • Visualização otimizada para Desktop e Mobile.

Pontos negativos

  • Somente médias são inseridas no boletim;
  • Não foi implementado um sistema para converter o boletim para PDF;
  • Caso precise renomear uma matéria, todas as notas serão redefinidas;
  • Para uma performance estável, é necessária uma conexão à internet de, no mínimo, 1 MB/s.
